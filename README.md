# taobao-cart
这是一个关于淘宝购物车的demo，数据皆为静态数组数据模拟。
开发环境：gulp + less + javascript。
下载后，首先安装依赖。
<pre>npm install</pre>

依赖安装完成后，启动服务(http://localhost:3000)
<pre>gulp</pre>
